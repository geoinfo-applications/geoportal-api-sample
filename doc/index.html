Geoportal Iframe API

Usage

Include API Script from Geoportal. This is evergreen and will be kept backwards compatible.

<script src="https://www.geoportal.ch/iframeApi.min.js"></script>

Wait for resources to be loaded, either by the means of your framework or VanillaJS:

<script>
    (function () {
        "use strict";
        window.addEventListener("load", function () {

            // Iframe API is loaded and ready, start Geoportal
            GeoportalAPI.iframeApi.register({
                iframeContainer: document.querySelector(".geoportal-container"),
                origin: "myapplication",
                primaryRegion: "myregion",
                mapId: 34,
                host: "https://www.geoportal.ch/"
            });

        }, true);
    }());
</script>

For everything available in GeoportalAPI.iframeApi see the API Docs for GeoportalIframeApi.

More examples see our Github Examples repository.
